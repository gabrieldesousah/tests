# Chrome Toolkit

Coleção de extensões do Google Chrome para apoiar desenvolvimento front-end.

## Extensões

| Extensão | Pasta | Descrição |
|----------|-------|-----------|
| BG/Text Color Inspector | [`extensions/bg-text-color-inspector`](extensions/bg-text-color-inspector) | Exibe classes de cor `bg-*` e `text-*` no hover |

## Instalação

1. Abra `chrome://extensions`
2. Ative **Modo do desenvolvedor**
3. Clique em **Carregar sem compactação**
4. Selecione a pasta da extensão desejada dentro de `extensions/`
